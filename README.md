# Echoes

> Every song has a memory.

---

## Philosophy

Echoes is not a music player.

It is not a Spotify clone.

Echoes is a safe place.

It is a personal journal where music becomes memories.

The goal is to create a calm, intimate and safe place where users can revisit the moments that songs represent.

Every feature should support that philosophy.

---

## About

Echoes is a personal music journal where every song tells a story.

This project is being built as a learning journey to master modern frontend development while creating a beautiful and meaningful web application.

---

---

## Core Features

- Create memories linked to songs
- Search songs using a music API
- Automatically retrieve album artwork
- Organize memories by date
- Explore memories through a calendar
- Filter memories by mood
- Listen to song previews (where available)
- Share beautiful memory cards
- Dynamic colors based on album artwork
